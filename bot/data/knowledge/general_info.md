# Общая информация о системах маркировки

## МДЛП (Мониторинг Движения Лекарственных Препаратов)
- Оператор: ЦРПТ (Центр развития перспективных технологий)
- Назначение: прослеживаемость лекарственных препаратов от производителя до пациента
- Основание: ФЗ-425 от 28.12.2017, ПП РФ №1556 от 14.12.2018
- Система: ИС МДЛП (mdlp.crpt.ru)
- Обязательна с: 01.07.2020
- Код маркировки: Data Matrix, содержит SGTIN (01+GTIN+21+серийный номер) + криптохвост

## ГИС МТ (Честный ЗНАК)
- Оператор: ЦРПТ
- Назначение: прослеживаемость различных товарных групп
- Система: markirovka.crpt.ru / честныйзнак.рф
- Товарные группы: табак, обувь, одежда, молочная продукция, вода, пиво, БАДы, мед. изделия, парфюмерия, косметика, ТСР, шины и др.

## Код маркировки (Data Matrix)
Структура кода для фармы и БАД:
- 01 + GTIN (14 цифр) — код товара
- 21 + серийный номер (до 13 символов) — уникальный ID экземпляра
- 91 + ключ проверки (4 символа)
- 92 + код проверки (до 44 символов) — криптографическая подпись

Пример: 010460043993125621ABC12345678\x1D91EE92\x1Dbase64crypto...

## Основные участники
- **Производитель** — заказывает коды, наносит на продукцию, вводит в оборот
- **Импортёр** — вводит в оборот при ввозе
- **Дистрибьютор** — передаёт/принимает через ЭДО
- **Розница/Аптека** — выводит из оборота при продаже через кассу
- **Медорганизация** — выводит при использовании/отпуске

## Разрешительный режим
С 2024 года на кассах действует «разрешительный режим»:
- При сканировании кода касса проверяет его статус в системе
- Если код невалидный/выведен — продажа блокируется
- Работает в онлайн-режиме, с офлайн-режимом при отсутствии интернета

## ЭДО (электронный документооборот)
- Обязателен для передачи маркированных товаров
- Операторы ЭДО: Контур.Диадок, СБИС, Такском, ЭДО Лайт и др.
- Формат: УПД (универсальный передаточный документ) с кодами маркировки
- ЭДО Лайт — бесплатный сервис от ЦРПТ для малого бизнеса
