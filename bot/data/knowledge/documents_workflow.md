# Документооборот в системе маркировки

## Основные операции и документы

### Ввод в оборот
- **Производитель**: подаёт сведения о нанесении кода маркировки и вводе в оборот
- **Импортёр**: подаёт сведения о ввозе и вводе в оборот после таможенного оформления
- Документ: уведомление о вводе в оборот

### Отгрузка / Приёмка
- Отправитель формирует УПД (универсальный передаточный документ) в системе ЭДО
- Получатель подтверждает приёмку в системе ЭДО
- При расхождениях — формируется акт расхождения
- В УПД указываются коды маркировки (SGTIN) или коды агрегатов (SSCC)
- При ОСУ (объёмно-сортовой учёт): в УПД указывается GTIN + количество

### Вывод из оборота
Способы вывода:
1. **Розничная продажа** — через кассу (ККТ), сканирование кода при продаже
2. **Списание** — уведомление о выводе по причине (истечение срока годности, брак, утрата)
3. **Отпуск по льготным рецептам** — для лекарств
4. **Использование в медицинской организации** — для лекарств и мед. изделий
5. **Экспорт** — при вывозе из РФ

### Возврат от покупателя
- При возврате товар возвращается в оборот
- Формируется документ «Возврат в оборот»
- Код маркировки снова становится активным

### Перемаркировка
- При повреждении кода Data Matrix
- Заказывается новый код, наносится на товар
- Подаётся уведомление о перемаркировке с указанием старого и нового кода

### Агрегация / Разагрегация
- **Агрегация**: объединение индивидуальных кодов (SGTIN) в групповую упаковку (SSCC)
- **Разагрегация**: извлечение из групповой упаковки
- Подаётся уведомление об агрегации/разагрегации

## Сроки подачи сведений

### Лекарственные препараты (МДЛП)
- Подтверждение операций: не позднее 1 рабочего дня
- Вывод из оборота через кассу: в момент продажи (онлайн)

### БАДы
- Подтверждение приёмки через ЭДО: 3 рабочих дня
- Вывод из оборота: в момент продажи через кассу

### Медицинские изделия
- Подтверждение приёмки через ЭДО: 3 рабочих дня
- Вывод из оборота: в момент продажи через кассу

### ТСР
- Подтверждение приёмки через ЭДО: 3 рабочих дня
- Вывод из оборота: в момент продажи или использования

## Особенности ОСУ (объёмно-сортовой учёт)
- Передача данных в разрезе GTIN + количество (без указания индивидуальных кодов)
- Используется на этапе перехода до полного поэкземплярного учёта
- В УПД обязательно указывается код GTIN и количество товара
