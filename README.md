# SGTIN AI 01-21 tool

Десктопное приложение для пакетной обработки SGTIN кодов маркировки.

## Возможности

- Верхнее поле — ввод SGTIN, по одному в строке (вставка/копирование/удаление/редактирование).
- Кнопка **«УДАЛИТЬ AI 01-21»** — убирает идентификаторы применения `01` и `21` из строк длиной **31 символ**.
  Иначе для конкретной строки выводится ошибка.
- Кнопка **«Добавить AI 01-21 (93 тоже)»** — добавляет `01` и `21` к строкам длиной **27** символов (GTIN+Serial13) или **20** символов (GTIN+Serial6, формат 93-кодов).
  Иначе для конкретной строки выводится ошибка.
- Кнопка **«93»** — для кодов вида `01+GTIN(14)+21+Serial+`*разделитель*`+93...` отрезает всё начиная с AI(93), оставляя только `01+GTIN+21+Serial`. Поддерживает Serial длиной 6 и 13 символов, разделитель GS или пробел (в т.ч. слитный код без разделителя).
- Кнопка **«Добавить 93»** — то же, что «Добавить AI 01-21», для коротких 93-кодов (вставляет `01`/`21` в `GTIN(14)+Serial(6)`).
- Нижнее поле — результат (вставка/копирование/удаление/редактирование).
- Кнопка **«Выгрузить в Excel»** — сохраняет содержимое нижнего поля в `.xlsx`.
- Поддержка тысяч строк, ошибки не прерывают обработку — корректные строки попадают в результат, проблемные перечисляются отдельно.

## Формат SGTIN

| Длина | Содержимое                                  |
|-------|---------------------------------------------|
| 31    | `01` + GTIN(14) + `21` + Serial(13)         |
| 27    | GTIN(14) + Serial(13)                       |
| 24    | `01` + GTIN(14) + `21` + Serial(6)  (93-формат) |
| 20    | GTIN(14) + Serial(6)  (93-формат)           |

При создании DataMatrix полный код маркировки (`01...21...` + AI `91`/`92`/`93`)
автоматически получает разделители GS1 (`\x1d`): либо по уже стоящим пробелам,
либо по структуре слитного кода. Если код содержит доп. AI, но разобрать
структуру не удалось, такая строка пропускается с ошибкой (чтобы не создать
нечитаемый DataMatrix). Префикс `[DATAMATRIX (GS1)]:` вырезается везде.

**FNC1 / GS1 DataMatrix.** Под кнопкой «Создать DataMatrix» есть галочка
**«GS1 DataMatrix (добавлять FNC1)»**, включённая при каждом запуске. Когда она
включена, в начало кода добавляется спецсимвол FNC1 (codeword 232) — это
настоящий GS1 DataMatrix, рекомендованный для Честного знака. Если галочку
снять, генерируется обычный DataMatrix без FNC1.

**Вставка base64.** Если в нижнее поле вставить готовую base64-строку, внутри
которой закодирован код маркировки, программа при генерации DataMatrix
автоматически распознаёт base64, декодирует его в код и строит картинку.
Обычные коды (начинающиеся с `01`+GTIN) за base64 не принимаются.

## Запуск из исходников

```bash
pip install -r requirements.txt
python sgtin_tool.py
```

## Сборка `.exe`

GitHub Actions автоматически собирает `.exe` при пуше в `main` и для каждого PR.
Готовый файл доступен:

1. Откройте вкладку **Actions** в репозитории.
2. Выберите последний успешный запуск workflow **Build Windows EXE**.
3. Скачайте артефакт **sgtin-tool-windows** — внутри `sgtin-tool.exe`.

Локальная сборка под Windows:

```powershell
pip install -r requirements.txt pyinstaller
pyinstaller --noconfirm --onefile --windowed --name sgtin-tool sgtin_tool.py
```

Файл появится в `dist\sgtin-tool.exe`.
