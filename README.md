# МаркировкаГид — AI-помощник по маркировке товаров

Телеграм-бот с AI-поддержкой для работы с системами маркировки МДЛП и ГИС МТ (Честный ЗНАК).

## Возможности

- 📋 **Таблица сроков маркировки** — актуальные даты вступления для всех товарных групп (лекарства, БАДы, мед. изделия, косметика, масла, ТСР)
- ❓ **AI-ответы на вопросы** — документооборот, штрафы, сроки, технические вопросы
- ➕ **Добавление данных** — расширение базы знаний через бота (для админов)
- 📊 **Краткая сводка** — быстрый обзор статуса по всем группам

## Установка

### 1. Клонирование и зависимости

```bash
git clone https://github.com/llCoolGamell/sgtin-ai-tool.git
cd sgtin-ai-tool
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

```bash
cp .env.example .env
```

Заполните `.env`:
- `TELEGRAM_BOT_TOKEN` — токен от @BotFather
- `OPENAI_API_KEY` — ключ OpenAI API (для AI-ответов)
- `ADMIN_IDS` — Telegram ID администраторов (через запятую)

### 3. Запуск

```bash
python run.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/table` | Таблица сроков маркировки (с выбором группы) |
| `/summary` | Краткая сводка по всем группам |
| `/ask <вопрос>` | Задать вопрос AI-помощнику |
| `/add` | Добавить запись в базу знаний (только админы) |
| `/help` | Показать справку |

Также можно просто написать вопрос текстом — бот ответит через AI.

## База знаний

Данные хранятся в двух форматах:
- **deadlines.json** — структурированная таблица сроков (JSON)
- **knowledge/*.md** — текстовые статьи по темам (Markdown)

### Обновление сроков
Отредактируйте `bot/data/deadlines.json` — добавьте новые этапы или измените статусы.

### Добавление статей
Через бота (команда `/add`) или вручную — создайте `.md` файл в `bot/data/knowledge/`.
